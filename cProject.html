Other than doing Alt+F, X to get File > Exit command, you can press Alt key then use the cursor keys to navigate edit.com menu system. The (possibly) expected function of using the mouse to click on the File command does not work by default. This is because the DOS Window properties have "Edit Options" enabled for "QuickEdit Mode" and "Insert Mode". Click on the top left of the DOS Window then "Properties", and then disable those two "Edit Options", and you will be able to use the mouse to execute commands on the edit.com menu bar.
